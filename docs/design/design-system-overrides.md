# EILA PSU Design System (Overrides)

## Core Retained Elements
- **Pattern:** Enterprise Gateway
- **Colors:**
  - Primary: `#1C1917` (Dark warm gray)
  - CTA: `#CA8A04` (Gold/Yellow)
  - Background: `#FAFAF9` (Off-white/Light mode base)
- **Typography:** EB Garamond + Crimson Text

## Overridden Style Constraints
- **Style Vibe:** "Clean Academic Professional" (Replaces "Vibrant & Block-based")
  - *Keywords:* spacious, institutional, minimal, trustworthy, elegant.
  - *Avoid:* bold, energetic, playful, block layout, geometric shapes.
- **Spacing:** Section gaps reduced to **32-40px** (less aggressive).
- **Typography Sizing:** Max **24-28px** for body headers (reduced from 32px+).
- **Animations:** Subtle fade-in only (Replaces animated patterns).
- **Interactions:** Gentle opacity or underline transitions on hover (Replaces bold color shifts).
